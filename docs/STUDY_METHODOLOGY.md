# 📚 Metodologia de Estudos — Python Fluente Lab

> Documento vivo.
>
> Sempre que nossa forma de estudar evoluir, este documento deverá ser atualizado para refletir a metodologia mais recente.

---

# 🎯 Objetivo

Construir um conhecimento sólido em Python através da combinação de:

- estudo teórico;
- prática constante;
- documentação;
- experimentação;
- revisão;
- versionamento com Git.

O objetivo não é apenas terminar um livro, mas construir um laboratório permanente de aprendizado.

---

# 🧠 Papéis de cada ferramenta

## 📖 Livro (Python Fluente)

É a principal fonte de conhecimento.

Durante a leitura devemos:

- entender os conceitos;
- identificar boas práticas;
- observar a filosofia da linguagem;
- anotar dúvidas.

O livro nunca será substituído.

---

## 👨‍🏫 ChatGPT

Responsável pela parte didática.

Durante cada aula deve:

- explicar conceitos;
- responder dúvidas;
- criar analogias;
- relacionar o conteúdo com Data Science;
- criar exercícios;
- revisar soluções;
- resumir os pontos importantes;
- produzir material em Markdown para revisão.

O ChatGPT é o professor da aula.

---

## 🤖 Codex

Responsável pela implementação.

Durante os estudos deve:

- criar notebooks;
- gerar exemplos;
- escrever scripts;
- organizar arquivos;
- refatorar código;
- criar documentação;
- localizar funções;
- sugerir melhorias;
- automatizar tarefas repetitivas.

O Codex é o engenheiro do laboratório.

---

# 🔄 Fluxo de Estudos

Cada seção do livro deverá seguir este ciclo.

## 1. Leitura

Ler cuidadosamente a seção do livro.

Durante a leitura:

- destacar dúvidas;
- identificar conceitos novos;
- observar exemplos importantes.

---

## 2. Aula

Discutir o conteúdo com o ChatGPT.

Objetivos:

- aprofundar conceitos;
- entender o "porquê";
- relacionar com aplicações reais.

---

## 3. Implementação

Solicitar ao Codex que transforme os conceitos em código.

Exemplos:

- notebooks;
- scripts;
- pequenos projetos;
- experimentos.

---

## 4. Experimentação

Executar o código.

Modificar.

Quebrar.

Testar.

Comparar resultados.

Aprender fazendo.

---

## 5. Revisão

Ao final da seção produzir:

- resumo;
- regras para memorizar;
- aplicações em Data Science;
- exercícios;
- autoavaliação.
